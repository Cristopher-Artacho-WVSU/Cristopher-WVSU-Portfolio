# Portfolio Website

This portfolio website was created to showcase my skills as part of my application to MATA Technologies. The development took place from **January 7 to January 10**.

---

## Tech Stack

- **EmailJS** – Email functionality
- **Firebase** – Realtime NoSQL Database
- **framer-motion** – Animation
- **lucide-react** – Icons
- **React** – Frontend Library
- **TailwindCSS** – Layout and Design
- **Node.js** – JavaScript runtime and package management

## Setup Instructions

- Clone the repository: git clone https://github.com/Cristopher-Artacho-WVSU/Cristopher-WVSU-Portfolio.git

- Navigate to the project directory: cd Cristopher-WVSU-Portfolio
- Install dependencies: npm install
- Run a localhost: npm run dev
- Copy and paste the link given
